# Threads
A web platform for upcycling and creative fashion. Transform clothing sustainably with AI-powered design tools and connect with talented artists.
Features
AI Studio – Visualize upcycling ideas with AI-generated patterns
Marketplace – Browse unique upcycled fashion pieces
Artist Profiles – Connect with creators and view portfolios
The Graveyard – Source materials and parts 

Tech Stack
React • Vite • Tailwind CSS • Framer Motion • React Router • Supabase
Quick Start
bash# Clone and install
git clone <repository-url>
cd re-thread
npm install

Run development server
npm run dev

Visit http://localhost:5173
